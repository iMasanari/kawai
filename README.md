# Sound Comic
先輩の卒業制作のプログラミング部分を手伝いました。
[TOXIC](https://imasanari.github.io/sound-comic/)

## 何これ
- BGM付きのwebコミック
- スクロールでページが切り替わると、音が変化していく
- スマートフォン対応（マナーモードの場合、音は鳴らない）

## 主に使用した技術
- HTML5 / CSS3
- TypeScript


- Web Audio API
- History API & HashChange
